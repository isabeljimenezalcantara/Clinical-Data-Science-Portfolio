# Clinical Data Science Portfolio

**Author:** Isabel Jimenez Alcantara  
**Objective:** Showcase practical skills in clinical data analysis, predictive modeling, and survival/causal analysis using real and public clinical datasets.

---

## Overview

This repository contains **three projects** that demonstrate key competencies for a Junior Clinical Data Scientist:

1. **Project 1: EDA of PERG-IOBA Dataset**  
   - Exploratory Data Analysis (EDA) of clinical and electrophysiological data from patients with ocular conditions.  
   - Focus on demographics, visual acuity, diagnoses, and clinical subgroups.

2. **Project 2: Predictive Modeling (PTB-XL / BIDMC CHF)**  
   - Development of machine learning models to predict clinical outcomes.  
   - Includes data preprocessing, feature engineering, model training, evaluation, and interpretation.

3. **Project 3: Survival and Causal Analysis (BIDMC CHF / COVID-19-MS)**  
   - Analysis of patient survival and causal relationships.  
   - Kaplan-Meier curves, Cox proportional hazards models, and causal inference exploration.

## Repository Structure
Clinical-Data-Science-Portfolio/

├── Proyecto_1_PERG_EDA/

├── Proyecto_2_Predictivo/

├── Proyecto_3_Supervivencia_Causalidad/

└── README.md


Each project folder contains:
- `notebooks/` – Colab notebooks for analysis
- `scripts/` – Reusable functions for plotting, preprocessing, or modeling
- `data/` – Processed datasets (CSV)
- `figures/` – Visualizations exported from notebooks
- `README.md` – Project-specific description

---

## How to Use

1. Open any notebook directly in **Google Colab** via the GitHub link:  
   `File → Open Notebook → GitHub → Enter Repository URL`
2. Run cells sequentially.  
3. Figures and processed data will be saved in the respective project folders for reproducibility.
